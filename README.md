# Real-Time Crypto Trading Dashboard

**Role:** Frontend Developer  
**Tech Stack:** React.js / WebGL / Three.js / Tailwind CSS / WebSockets  

## Overview
This is a **high-performance real-time crypto trading dashboard** built to visualize live cryptocurrency data with **WebGL-powered charts** and **sub-millisecond updates**.  

Key highlights:  
- **Real-time streaming:** Sub-millisecond updates via WebSockets  
- **Interactive WebGL charts:** Smooth 3D data visualization  
- **Responsive and cinematic UI:** Modern, professional layout  
- **Portfolio & market overview cards:** Quick glance at key metrics
